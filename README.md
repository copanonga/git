## Crear repositorio

```

echo "# Título" >> README.md
git init
git add .
git commit -m "Primer commit"
git remote add origin url...
git remote -v
git push -u origin master

```

## Comprobar versión Git

```

git --version

```

## Clonar repositorio

